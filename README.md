## 🎮 게임 시작 방법

프로그램을 실행하면 아래 순서대로 게임이 시작됩니다.

1. **이름 입력**
   - 사용자 이름을 입력합니다.
   - 빈 값은 입력할 수 없습니다.

2. **주량 선택**
   - 1~5번 보기 중 자신의 주량을 선택합니다.
   - 선택한 주량에 따라 치사량이 정해집니다.
   - 잘못된 번호를 입력하면 다시 선택해야 합니다.

3. **함께 게임할 친구 초대**
   - 함께 대결할 컴퓨터 참가자 수를 입력합니다.
   - 최소 1명, 최대 3명까지 초대할 수 있습니다.
   - 초대한 컴퓨터 참가자의 이름과 주량은 랜덤으로 정해집니다.

4. **현재 술자리 상황 확인**
   - 각 참가자가 지금까지 몇 잔을 마셨는지 확인할 수 있습니다.
   - 치사량까지 몇 잔이 남았는지도 함께 출력됩니다.

5. **게임 선택**
   - 게임 목록에서 원하는 게임 번호를 입력합니다.
   - 사용자가 게임을 선택한 뒤에는 컴퓨터 참가자들도 차례대로 랜덤하게 게임을 선택합니다.

6. **게임 진행 및 종료**
   - 선택된 미니게임의 결과에 따라 참가자의 마신 잔 수가 증가합니다.
   - 누군가 치사량에 도달하면 최종 play 상황을 출력하고 게임이 종료됩니다.

## 지수의 게임: 더 게임 오브 데스

### 게임 설명

참가자들이 각자 한 명을 가리키고, 술래가 외친 숫자만큼 손가락 방향을 따라 이동하는 게임입니다.  
숫자가 끝나는 지점에 있는 참가자가 최종 당첨자가 되며 벌주를 마십니다.

### 진행 방식

1. 게임이 시작되면 각 참가자는 한 명을 가리킵니다.
2. 사용자는 직접 가리킬 사람을 선택합니다.
3. 컴퓨터 참가자들은 가리킬 사람을 랜덤으로 선택합니다.
4. 술래는 사용자가 직접 지정합니다.
5. 술래는 임의의 숫자를 하나 외칩니다.
6. 술래부터 시작하여, 각 참가자가 가리키는 방향을 따라 숫자를 셉니다.
7. 술래가 외친 숫자에 도착한 참가자가 최종 당첨자가 됩니다.
8. 최종 당첨자는 벌주 1잔을 마십니다.

### 벌칙 규칙

- 최종 당첨자: 벌주 1잔

## 예원의 게임: 베스킨라빈스 31

### 게임 설명

참가자들이 돌아가면서 숫자를 이어 말하는 게임입니다.  
각 차례마다 1~3개까지 숫자를 말할 수 있으며, 31을 말하는 사람이 벌주를 마십니다.

### 진행 방식

1. 현재 숫자에서 이어서 숫자를 말합니다.
2. 사용자는 자신의 차례에 말할 숫자 개수를 1~3개 중에서 선택합니다.
3. 컴퓨터 참가자들은 말할 숫자 개수를 1~3개 중 랜덤으로 선택합니다.
4. 선택한 개수만큼 숫자를 이어 말합니다.
5. 숫자 31을 말한 참가자가 벌주를 마십니다.
6. 게임이 끝나면 벌주 대상자의 마신 잔 수가 1잔 증가합니다.

### 벌칙 규칙

- 31을 말한 참가자: 벌주 1잔

## 연우의 게임: 소수파 술자리 밸런스 점수 게임

### 게임 설명

총 3개의 밸런스 질문에 답하는 게임입니다.  
각 질문마다 참가자들은 A 또는 B 중 하나를 선택합니다.  
선택 결과에 따라 다수파와 소수파를 나누고, 점수를 계산합니다.

### 진행 방식

1. 밸런스 질문과 A/B 선택지가 출력됩니다.
2. 사용자는 A 또는 B를 직접 입력합니다.
3. 컴퓨터 플레이어들은 A 또는 B 중 하나를 랜덤으로 선택합니다.
4. 모든 참가자의 선택을 공개합니다.
5. 다수파와 소수파를 판정하고 점수를 반영합니다.
6. 총 3라운드를 진행한 뒤 최종 점수를 확인합니다.
7. 최종 점수가 가장 낮은 참가자가 벌칙주 1잔을 마십니다.
8. 최저 점수자가 여러 명이면 해당 참가자들이 모두 1잔씩 마십니다.

### 점수 규칙

- 다수파: +1점
- 소수파: -1점
- 연속 소수파: 추가 -1점
- 동률: 모두 0점

### 예시

참가자가 4명일 때 A를 3명, B를 1명이 선택하면 B를 선택한 사람이 소수파가 됩니다.  
소수파는 -1점, 다수파는 +1점을 얻습니다.  
같은 사람이 다음 라운드에도 소수파가 되면 연속 소수파로 추가 -1점을 받아 총 -2점이 됩니다.

### 벌칙 규칙

- 최종 점수가 가장 낮은 참가자: 벌칙주 1잔
- 최저 점수자가 여러 명일 경우: 해당 참가자 전원 벌칙주 1잔씩
